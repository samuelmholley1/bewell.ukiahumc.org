# Be Well Center — Brand Guide

> Warm • Trustworthy • Calm • Nature-rooted • Human • Accessible
> Avoid: techy, sterile, trendy, corporate

---

## Fonts

### Primary Serif — Headings & Logo

**Cormorant Garamond** (weight 500–700)
Use for: H1, H2, logo text, printed titles
Gives: warmth + dignity without looking old

### Body Font — Content Text

**Source Serif 4** (weight 400–600)
Use for: paragraphs, brochures, long text, letters
Chosen because: extremely readable for seniors

### UI Sans — Interface Elements

**Source Sans 3** (weight 400–600)
Use for: buttons, forms, nav, captions, labels
Keeps interface clean and accessible

### CSS Stack

```css
--font-family-display: "Cormorant Garamond", serif;
--font-family-serif: "Source Serif 4", Georgia, serif;
--font-family-sans: "Source Sans 3", Arial, sans-serif;
```

---

## Colors

### Primary Green (brand anchor) — `#556B2F`

Headings, logo text, buttons, links

### Secondary Warm Gold — `#D99A2B`

Highlights, donation buttons, accents, dividers

### Soft Nature Green — `#8FAE4A`

Backgrounds, cards, hover states, subtle accents

### River Blue Accent — `#6CA6C1`

Links, informational highlights, icons (use sparingly)

### Neutral Warm Background — `#F7F3EA`

Page background, section panels, printable docs

### Dark Text Color — `#2B2B2B`

Use instead of pure black (softer for older eyes)

### CSS Colors

```css
--color-primary: #556B2F;
--color-secondary: #D99A2B;
--color-accent: #8FAE4A;
--color-accent-blue: #6CA6C1;
--color-neutral-bg: #F7F3EA;
--color-text: #2B2B2B;
```

---

## Typography Scale (senior-friendly)

```
body:    18px minimum (1.125rem)
h1:      42px (2.625rem)
h2:      32px (2rem)
h3:      26px (1.625rem)
buttons: 18–20px
line-height: 1.6+
```

Never go below **16px** anywhere.

---

## UI Style Rules

- Rounded corners (6–10px / 0.375–0.625rem)
- Soft shadows only
- Big buttons
- Plenty of spacing
- Avoid thin fonts
- Avoid all-caps paragraphs

---

## Brand Voice

Warm, plainspoken, reassuring.

**Use:** short sentences, real words (not marketing fluff), supportive tone, clear calls to action

**Avoid:** corporate language, tech jargon, exaggerated claims

---

## Naming Convention

- **Brand title / label:** Be Well Center
- **In running text / descriptions:** the Be Well Center

---

## One-Line Summary for Copilot

> Use Cormorant Garamond for headings, Source Serif 4 for body, Source Sans 3 for UI. Colors: olive green #556B2F primary, warm gold #D99A2B accent, soft natural palette, large readable typography, warm nonprofit tone.
